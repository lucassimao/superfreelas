# SuperFreelas - Local and fast jobs made easy
---

## Goal
  To build a marketplace of local available microtasks and temporary jobs in order to help people to make money as fast as possible

## Tech stack
    - Backend
      - NodeJS
        - expressJS
        - mongoose
        - pug

    - Frontend
      - React

    - Database
      - MongoDB

    - Tests
      - supertest
      - jest
